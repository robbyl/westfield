The NEW Brady BMP71 Printer is now available
The BMP71 Mobile Label Printer is Brady's newest high end portable cable and wire label printer. As well as being the ideal cable and wire printing solution for the telecom/datacom and electrical markets, the BMP71 is also perfect for printing health and safety signs and labels in the process industries. The BMP71 is a user friendly printer with a large full colour LCD display, an easy to use menu driven user interface, and a full QWERTY keyboard layout making it quick and easy to create labels using text, symbols and coding.
Features:
  • The BMP71 is filled with smart features, inbuilt application settings for telecom/datacom and safety labels
  • WYSIWYG user interface, no need for print preview, no label waste, every label on the roll is printed
  • High print speed of 38.1mm/sec (1.5 ips)
  • Available with full QWERTY, AZERTY, QWERTZ, or CYRILLIC keyboard layout
  • Multiple languages & character sets for localised use in different regions
  • Easy loading of label and ribbon consumables
  • Smart touch cell technology with even more smart parameters included to speed up and simplify usage of the printer
  • Long lasting battery allowing you to print labels anywhere you need them
  • On screen indicators for label size, battery level
  • Auto font sizing, 200MB memory allowing storage of 100,000 labels

The BMP71 is capable of printing a variety of label formats including PermaSleeves™, Self-laminating Labels, EPREP, Diecut Labels and much more...

All these materials contain a smart cell which automatically selects the correct printer settings avoiding mistakes and eliminating label waste. BMP71 is supplied with:
  • BMP71 printer Battery pack
  • AC adapter/battery charger
  • USB connection cable
  • TLS 2200®/HandiMark™ label roll adapter
  • Cleaning card and swab
  • Product CD (contains user manual, print driver and tutorial)
  • Quick Start Guide
  • M71-R6000 black ribbon
  • M71-31-423 diecut labels and M71C-2000-580-WT labels
  • Hard carry case
